## 파일 시스템
### pathlib : 파일 시스템 경로를 객체 지향적으로 다룰 수 있게 해주는 파이썬 표준 라이브러리
- pathlib의 path 클래스 우리가 편하려고 쓴다
- cwd, home, 서픽스(특정 경로)
- 
```python
Path('디렉토리').mkdir(exist_ok = True) #True로 하면디렉토리가 이미 있어도 오류를 내지 않음
Path('파일명').write.text("233232") # 새로운 파일명에 텍스트를 넣어줌
```
```python
with Path('dqwd').open('a', encoding = 'utf-8') as file: # 기존 내용에 추가 
    file.write('\n')
    #........
```
```python
from pathlib import Path

current_path = Path.cwd()
```
